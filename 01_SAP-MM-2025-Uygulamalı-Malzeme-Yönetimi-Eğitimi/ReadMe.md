# Malzeme Kodu Değiştime ve Raporlama 
 
 + İlk olarak malzeme üretebilmek için ham maddeye ihtiyacımız var. Bu ham maddeleri, yarı mamül ve bitmiş ürünlere dönüştürerek processimizi tamamlamış oluyoruz. 

+ Bu doğal akış, SAP üzerinde de aynı sırayla devam etmektedir.

+ Yani sistem üzerinde madde kodları, yarı mamül kodları ve bitmiş ürün kodlarının tanımlanmış olması gerekiyor.

+ Transaction: **MM02** - İşlenmiş malzeme kodunun değiştirilmesi

+ **MİP:** Malzeme İhtiyaç Planı

+ Transaction: **MM03** - İşlenmiş malzeme kodunun görüntülenmesi

+ Transaction: **MM60** - Malzeme Listesi Raporlama
